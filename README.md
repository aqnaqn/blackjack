# blackjack
An in-browser Javascript Blackjack game with options to specify the starting bankroll amount, the blackjack pay-off (3:2, 6:5, and 1:1 (!) ), and the number of decks, with extensive keyboard shortcuts

This was created using claude.ai (!).  The nitial prompt and subsequent refinement prompts are in blackjack.html's commit messages.

Tested on Chrome, Safari, and Firefox.

## Quick Start
Load blackjack.html in browser.

Keyboard shortcuts:
* d deal
* h hit
* s stand
* t double
* p split
* l double for less (will be prompted for amount)
* 2 add $25 to current bet
* 5 add $500 to current bet
* 1 add $100 to current bet
